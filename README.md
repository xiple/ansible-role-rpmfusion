Ansible role: RPM Fusion
=========

An ansible role that installs RPM Fusion free & nofree repositories.

Requirements
----------------

None.

Role Variables
----------------

None.

Supported distributions
----------------

This role has been been developed and tested on the following distributions :

- Fedora 42

Example Playbook
----------------

```yaml
- hosts: all
  roles:
    - xiple.rpmfusion
```
